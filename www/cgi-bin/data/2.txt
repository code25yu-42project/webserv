## Directory Structure
* `srcs/` : C source code files for each project
* `docs/requirements.md` : Summary of requirements, implementation methods, and key concepts for each project
* `README.md` : Repository overview and usage instructions


## Project List & Key Topics
* **C 00** : Basic I/O, loops, conditionals, character/number processing
* **C 01** : Pointers, double pointers, swapping values, printing strings/length, reversing/sorting arrays
* **C 02** : String copying/checking/conversion, string property detection, case conversion
* **C 03** : String comparison/concatenation/searching, length-limited operations, substring handling, safe string manipulation
* **C 04** : String length/printing, number conversion/output, base conversion, custom `atoi`
* **C 05** : Mathematical recursion/iteration (factorial, power, Fibonacci, primes, square root, backtracking)
* **C 06** : Command-line arguments, string output, sorting
* **C 07** : Dynamic memory allocation, string/array creation, string join/split, base conversion
* **C 08** : Headers/macros/structs, struct arrays, memory management, output
* **C 09** : Static libraries, Makefiles, string splitting (`split`), automation
* **C 11** : Function pointers, advanced array/string handling, dynamic memory, sorting/counting

## How to Use
1. Browse each directory to learn and practice with the provided source files.
2. Refer to `docs/requirements.md` for project requirements and key concepts.
3. Some projects include Makefiles, shell scripts, and header files as well.

## 디렉토리 구조
- `srcs/` : 각 과제별 C 소스 코드 파일
- `docs/requirements.md` : 각 과제별 요구사항, 구현법, 핵심 개념 정리
- `README.md` : 저장소 소개 및 사용법

## 과제 목차 및 주요 주제
- **C 00** : 기초 입출력, 반복문, 조건문, 문자/숫자 처리
- **C 01** : 포인터, 다중 포인터, 값 교환, 문자열 출력/길이, 배열 뒤집기/정렬
- **C 02** : 문자열 복사/검사/변환, 문자열 속성 판별, 대소문자 변환
- **C 03** : 문자열 비교/연결/탐색, 길이 제한, 부분 문자열, 안전한 문자열 처리
- **C 04** : 문자열 길이/출력, 숫자 변환/출력, 진법 변환, atoi 구현
- **C 05** : 수학적 재귀/반복(팩토리얼, 거듭제곱, 피보나치, 소수, 제곱근, 백트래킹)
- **C 06** : 명령행 인자 처리, 문자열 출력, 정렬
- **C 07** : 동적 메모리 할당, 문자열/배열 생성, 문자열 합치기/분리, 진법 변환
- **C 08** : 헤더/매크로/구조체, 구조체 배열, 메모리 관리, 출력
- **C 09** : 정적 라이브러리, Makefile, 문자열 분리(split), 자동화
- **C 11** : 함수 포인터, 배열/문자열 고급 처리, 동적 할당, 정렬/카운팅

## 사용 방법
1. 각 디렉토리별로 소스 파일을 참고하여 학습 및 실습이 가능합니다.
2. `docs/requirements.md` 파일에서 각 과제별 요구사항과 핵심 개념을 확인할 수 있습니다.
3. 일부 과제는 Makefile, 쉘 스크립트, 헤더 파일 등도 포함되어 있습니다.
